# HarmonyComponent view 组件生成 schema

HarmonyView 是一个视图组件。
componentName 为渲染时候的组件名称，props 为组件绑定的属性，children 为子组件列表，id 为可以动态生成的唯一 id。

## view 组件生成 schema 代码示例

### 基本用法

```json
{
  "componentName": "HarmonyView",
  "props": {
    "className": "component-base-style",
    "hoverClass": "hover-style" // 可以设置按下去的样式类。
  },
  "children": [],
  "id": "b662a654"
}
```
